# 보담 콘텐츠 전략

## 1. 비즈니스 목표

```
검색 유입 → 콘텐츠 신뢰 구축 → 상담 전환 → 손해사정 수임
```

**수익 모델**: 손해사정 수수료 (보험금의 10~15%, 최저 50만원)

## 3. 콘텐츠 카테고리 & 필수 글 목록

### A. 보험금 청구 가이드 (How-to) - 20개
```
[실손보험]
- 실손보험 청구 방법 완벽 가이드 (2024)
- 실손24 앱 사용법 - 서류 없이 청구하기
- 실비보험 청구 서류 체크리스트
- 통원치료비 청구 - 얼마까지 받을 수 있나
- 입원비 청구 시 주의사항

[자동차보험]
- 교통사고 났을 때 보험 처리 순서
- 자동차보험 청구 서류 준비 방법
- 대인배상 vs 대물배상 차이점
- 렌터카 사고 보험 처리 방법
- 음주운전 피해자 보상 받는 법

[생명/건강보험]
- 암 진단비 청구 방법과 주의사항
- 수술비 보험금 청구 가이드
- 진단비 vs 수술비 vs 입원비 차이
- 후유장해 보험금 청구 절차
```

### B. 분쟁/거절 대응 - 15개
```
- 보험금 거절당했을 때 대처법 5단계
- 보험사 과소지급 의심될 때 확인할 것
- 의료자문 악용 사례와 대응 방법
- 고지의무 위반으로 거절? 아직 방법 있다
- 금융감독원 민원 신청 방법
- 분쟁조정위원회 활용 가이드
- 보험금 소송 전 알아야 할 것
```

### C. 합의금/보상금 계산 - 10개
```
- 교통사고 합의금 계산 방법 (2024 기준)

- 후유장해 등급별 보상금 기준
- 사망사고 보상금 산정 방법
```

## 4. 전환 퍼널 설계

### Level 1: 검색 유입
```
[검색어] "실손보험 청구 방법"
    ↓
[블로그 글] 실손보험 청구 완벽 가이드
```

### Level 2: 인게이지먼트 (체류시간 증가)
```
글 중간/하단 CTA:
├── "내 상황에서 얼마 받을 수 있을까?" → 자가진단 퀴즈
├── "청구 서류 체크리스트 다운로드" → 이메일 수집
└── "비슷한 사례 보기" → 분쟁사례 페이지
```

### Level 3: 전환 (상담 신청)
```
자가진단 결과:
├── "더 받을 수 있을 것 같아요" → 무료 상담 신청
├── "거절 가능성 있어요" → 전문가 검토 필요
└── "분쟁 중이시네요" → 손해사정사 상담 안내
```

## 5. 페이지별 CTA 전략

### 블로그 포스트
```html
<!-- 글 중간 (40% 지점) -->
<CTA type="soft">
  "혹시 보험금 청구가 거절되셨나요?"
  → 분쟁사례 페이지 링크
</CTA>

<!-- 글 하단 -->
<CTA type="hard">
  "전문가에게 무료로 검토받기"
  → 상담 신청 페이지
</CTA>
```

### 분쟁사례 페이지
```
각 사례 하단:
"비슷한 상황이신가요? 무료 검토 신청"
```

### 자가진단 퀴즈
```
결과 페이지:
├── 청구 가능: "지금 바로 청구하세요" + 가이드 링크
├── 추가 가능: "더 받을 수 있어요" + 상담 CTA
└── 분쟁 필요: "전문가 도움 필요" + 상담 CTA (강조)
```

## 6. 유입 도구 (리드 마그넷)

### 계산기
1. **교통사고 합의금 계산기**
   - 입력: 치료기간, 입원일수, 월소득, 과실비율
   - 출력: 예상 합의금 범위
   - CTA: "정확한 금액은 전문가 상담으로"

2. **보험금 예상 계산기**
   - 입력: 보험 종류, 진단명, 치료 내용
   - 출력: 예상 보험금 항목
   - CTA: "놓치는 항목 없는지 검토받기"

### 체크리스트 (PDF 다운로드)
1. 실손보험 청구 서류 체크리스트
2. 교통사고 시 꼭 해야 할 것 10가지
3. 보험금 거절 시 대응 체크리스트

### 자가진단 퀴즈 (강화)
1. "내 보험금, 제대로 받고 있나?" (5문항)
2. "교통사고 합의금, 적정한가?" (7문항)
3. "보험금 추가 청구 가능성 진단" (6문항)

## 10. 경쟁사 분석

### 시그널플래너
- 강점: 앱 기반, 카카오톡 연동, 보험 분석 기능
- 약점: 손해사정 전문성 부족
- 차별화: 손해사정 전문 콘텐츠, 분쟁 해결 사례

### 뱅크샐러드
- 강점: 콘텐츠 SEO 강함, 브랜드 인지도
- 약점: 보험 판매 중심, 분쟁 대응 약함
- 차별화: 실제 분쟁 해결 경험, 손해사정사 직접 상담

### 더바다/사고링크
- 강점: 합의금 계산기, 변호사 연결
- 약점: 콘텐츠 부족, 정보성 약함
- 차별화: 풍부한 정보 콘텐츠 + 계산기 + 상담

## 핵심 메시지

> "보험사가 알려주지 않는 것, 손해사정사가 알려드립니다"

- 보험사 편이 아닌 고객 편
- 거절당해도 방법이 있다
- 무료 상담, 성공 시에만 비용
